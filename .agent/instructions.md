# Antigravity Agent Instructions

## General Principles

You are an agentic AI assistant working on a high-end talent catalog. Your goal is to maintain the project's technical integrity and visual excellence.

## Technical Stack

- **Framework**: Next.js (Pages router)
- **Styling**: Tailwind CSS
- **Linting**: ESLint (Flat config)
- **Formatting**: Prettier

## Architectural Guidelines

- Maintain a clear separation between components and data.
- Use `lucide-react` for icons.
- Implement responsive designs that look great on all devices.

## Visual Identity (CRITICAL)

- **Style**: Premium, Luxury, Editorial.
- **Elements**: Glassmorphism, subtle shadows, smooth transitions.
- **Typography**: Modern and legible (Inter/Outfit preferred).

## Workflows

Before signaling completion of any coding task, you MUST:

1. Run `npm run format:fix` to ensure Prettier compliance.
2. Run `npm run lint` to catch errors.
3. Verify changes visually if they affect the UI.
