# Catalogo (estructura Next.js)

Este directorio contiene una estructura mínima para ejecutar una app Next.js.

Instrucciones rápidas:

1. Abrir terminal en este directorio:

```bash
cd Catalogo
```

2. Instalar dependencias:

```bash
npm install
```

3. Ejecutar en desarrollo:

```bash
npm run dev
```

Puntos importantes:
- El script `dev` ejecuta `next dev`. También se conservan los scripts de Vite como `vite:dev` si desea mantener la configuración previa.
- Si desea usar TypeScript u opciones avanzadas de Next.js, se deben añadir `tsconfig.json` y ajustar dependencias.
